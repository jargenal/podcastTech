# Voces de referencia

Coloca aqui tus archivos `.wav` de referencia para XTTS v2.

Nombres sugeridos:

- `default_es_mx.wav`
- `default_es_neutro.wav`
- `default_es_es.wav`

Recomendaciones:

- mono o stereo limpio
- 16 kHz a 48 kHz
- 8 a 20 segundos
- voz sin musica de fondo
- pronunciacion natural del dialecto objetivo
